## Usage

**Easily and quickly convert CIDR to IP address list**

```python
# convert CIDR to IP list
>>> import cidres
>>> cidres.cidr.convert("129.153.48.0/20")
```

## Install

**very easy to install, supports minimum version of python3.4**

```bash
pip install cidres
```
